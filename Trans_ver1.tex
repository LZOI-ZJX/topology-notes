\section{区间 (Intervals)}\label{ux533aux95f4-intervals}

在本节中，我们将回顾实直线 \(\mathbb{R}\) 的子集。

\subsection{实直线的区间 (Intervals of the real
line)}\label{ux5b9eux76f4ux7ebfux7684ux533aux95f4-intervals-of-the-real-line}

令 \(\overline{\mathbb{R}} = \mathbb{R} \cup \{\pm\infty\}\) 表示
\textbf{扩展实直线} (extended real line)。

\begin{definition}[区间]\protect\hypertarget{def-interval}{}\label{def-interval}

\(\mathbb{R}\) 的一个 \textbf{区间} 是指子集 \(I\)，满足对于任何
\(x, y \in I\)，只要 \(x < z < y\) 或 \(y < z < x\)，就有 \(z \in I\)。

令 \(\bar{a}, \bar{b} \in \overline{\mathbb{R}}\)，区间 \(I\)
是以下四种集合类型之一：

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  \(]\bar{a}, \bar{b}[ = \{x \in \mathbb{R} : \bar{a} < x < \bar{b}\}\);
\item
  \([\bar{a}, \bar{b}] = \{x \in \mathbb{R} : \bar{a} \leq x \leq \bar{b}\}\);
\item
  \([\bar{a}, \bar{b}[ = \{x \in \mathbb{R} : \bar{a} \leq x < \bar{b}\}\);
\item
  \(]\bar{a}, \bar{b}] = \{x \in \mathbb{R} : \bar{a} < x \leq \bar{b}\}\).
\end{enumerate}

\end{definition}

对于 \(\bar{a}, \bar{b} \in \overline{\mathbb{R}}\)，我们有：

\begin{itemize}
\tightlist
\item
  若 \(\bar{a} \geq \bar{b}\)，则
  \(]\bar{a}, \bar{b}[ = [\bar{a}, \bar{b}[ = ]\bar{a}, \bar{b}] = \emptyset\)；
\item
  若 \(\bar{a} > \bar{b}\) 或 \(\bar{a} = \bar{b} = \pm\infty\)，则
  \([\bar{a}, \bar{b}] = \emptyset\)；
\item
  若 \(\bar{a} = \bar{b} \in \mathbb{R}\)，则
  \([\bar{a}, \bar{b}] = \{\bar{a}\}\)。
\end{itemize}

\begin{definition}[空、退化与真区间]\protect\hypertarget{def-interval-types}{}\label{def-interval-types}

\(\mathbb{R}\) 的区间 \(I\) 如果是空集，则称其为 \textbf{空的} (empty)；
如果它只包含单个元素，则称其为 \textbf{退化的} (degenerate)；
如果它既非空也非退化，则称其为 \textbf{真的} (proper)。

\end{definition}
